# Gym Plan Web App

A mobile-friendly multi-page gym plan designed for iPhone, with a soft Pinterest-style theme and separate pages for each workout day.

## Pages

- `index.html` — Home
- `day1.html` — Glutes + Hamstrings
- `day2.html` — Back + Shoulders + Core
- `day3.html` — Glutes + Arms
- `day4.html` — Upper Body Shape + Core + Cardio
- `progress.html` — Progression guide
- `icon-180.png` — iPhone home screen icon

## Features

- iPhone-friendly layout
- soft beige / Pinterest-style design
- separate pages for each training day
- bottom navigation across pages
- YouTube demo links for each exercise
- Apple Home Screen icon support
- GitHub Pages ready
